## Descrição do projeto

Este projeto investiga a viabilidade de construir uma árvore de dependências de competências matemáticas para o ENEM com apoio de modelos de linguagem, dentro da perspectiva do framework SCAD (Skill-Centric Assessment Design).

Para isso, foram analisadas questões de matemática do ENEM, no período de 2009 a 2025, em conjunto com a matriz de referência dos objetos de aprendizagem de matemática. O Graphify foi utilizado (em conjunto com o Claude Code) para extrair relações entre conteúdos e estruturar um grafo de conhecimento. Em seguida, o Codex foi empregado para gerar uma árvore de dependências entre competências matemáticas.

O resultado produzido pelo modelo foi então comparado com uma estrutura elaborada por especialista, com o objetivo de avaliar se uma abordagem indutiva com LLMs pode apoiar a construção de hierarquias de competências educacionais de forma mais rápida, sistemática e menos subjetiva.
